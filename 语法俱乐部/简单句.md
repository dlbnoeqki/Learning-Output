## 五种单句的基本句型

|              |                   |          |
| ------------ | ----------------- | -------- |
| 1. S+V       | 主语 + 动词           |          |
| 2. S+V+O     | 主语 + 动词 + 宾语      | S：主语     |
| ==3. S+V+C== | ==主语 + 动词 + 补语==  | ==V：动词== |
| 4. S+V+O+O   | 主语 + 动词 + 宾语 + 宾语 | O：宾语     |
| ==5. S+V+O+C==   | ==主语 + 动词 + 宾语 + 补语== | ==C：补语==     |
主语：句子所描述的对象
动词：构成叙述的主要内容
